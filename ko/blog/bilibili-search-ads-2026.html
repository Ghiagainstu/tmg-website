Bilibili 검색 광고 2026: 완전 광고 가이드

Bilibili는 중국에서 가장 중요한 광고 플랫폼 중 하나로 부상했으며, 그 검색 광고 에코시스템은 그 성장의 최전선에 있습니다. 월간 활성 사용자 수 3억 4천만 명——그중 65퍼센트가 30세 미만——Bilibili는 중국의 디지털 환경에서 가장 집중된 Z세대 오디언스에 대한 접근을 광고주에게 제공합니다. 2026년, Bilibili의 검색 광고 제품은 크게 확장되어, 새로운 형식, 향상된 타겟팅 기능, 개선된 측정 도구가 도입되어 플랫폼에서의 검색 광고가 그 어느 때보다 강력하고 접근 가능해졌습니다.

이 포괄적인 가이드는 2026년 Bilibili 검색 광고에 대해 브랜드 광고주가 알아야 할 모든 것을 다룹니다——사용 가능한 형식과 타겟팅 옵션부터 최적화 전략 및 성과 측정까지.

2026년 Bilibili 검색 광고의 현황

Bilibili의 검색 광고 수익은 지난 2년간 크게 성장했으며, 이는 광고주 수요 증가와 검색 제품에 대한 플랫폼 투자를 반영합니다. 2026년 1분기에 Bilibili는 검색 광고 수익이 총 광고 수익의 18퍼센트를 차지했다고 보고했습니다——2024년 1분기의 12퍼센트에서 증가했습니다. 이 성장 궤적은 검색이 Bilibili의 수익화 전략에서 점점 더 중심적인 기둥이 되고 있음을 시사합니다.

이 성장을 이끄는 여러 거시적 트렌드가 있습니다:

왜 중요한가

Bilibili는 Z세대와 밀레니얼 세대에게 도달하는 중국 최고의 플랫폼. 독특한 콘텐츠 문화가 기회 창출.

  • 검색 행동의 확장: Bilibili의 총 검색 쿼리는 2026년 1분기에 전년 대비 31퍼센트 성장했으며, 사용자들이 비디오 콘텐츠, 제품 조사, 엔터테인먼트 발견을 위한 주요 검색 목적지로 플랫폼을 점점 더 많이 활용하고 있습니다.
  • 광고주 도입의 가속화: Bilibili의 활성 검색 광고주 수는 전년 대비 47퍼센트 증가했으며, 각 버티컬의 브랜드들이 의도의 순간에 젊은 소비자에게 도달하는 플랫폼의 고유한 능력을 인식하고 있습니다.
  • 새로운 형식의 등장: Bilibili는 2026년 여러 새로운 검색 광고 형식을 도입하여, 기존 검색 결과 배치를 넘어 유기적 콘텐츠와 유료 콘텐츠의 경계를 흐리게 하는 일시정지 광고, 인피드 검색 광고, 맥락적 배치를 포함하도록 확장했습니다.

Bilibili 검색 광고 형식: 완전한 개요

Bilibili에서 사용 가능한 검색 광고 형식의 전체 범위를 이해하는 것은 효과적인 광고 전략을 구축하는 데 필수적입니다. 다음은 각 형식에 대한 상세한 분석입니다.

표준 검색 결과 광고

표준 검색 결과 광고는 유기적 검색 결과 피드 내에 표시되며, 후원 콘텐츠로 명확하게 라벨링됩니다. 이 광고들은 비디오 썸네일, 텍스트 설명, 콜 투 액션 버튼을 지원합니다. Bilibili의 검색 광고 제공의 기반으로, 브랜드 인지도에서 직접 전환까지 virtually 모든 캠페인 목표에 적합합니다.

주요 사양:
- 검색 결과 내 1~5위에 표시
- 비디오, 이미지, 텍스트 크리에이티브 지원
- 최소 입찰가는 클릭당 0.5위안부터
- 평균 클릭률: 5.2퍼센트

브랜드 존

브랜드 존은 Bilibili의 프리미엄 검색 배치로, 사용자가 특정 브랜드 키워드를 검색할 때 검색 결과 페이지 상단을 차지합니다. 비디오, 인터랙티브 요소, 브랜드 채널이나 제품 페이지로의 직접 링크를 포함한 리치 미디어를 지원합니다.

주요 사양:
- 페이지 상단의 독점적 배치
- 최대 60초 비디오 지원
- 인터랙티브 요소와 미니 프로그램 통합 포함
- 평균 클릭률: 12.4퍼센트
- 키워드 독점성이 포함된 일별 과금 모델

일시정지 광고 (2026년 신규)

2026년에 도입된 Bilibili의 가장 혁신적인 검색 광고 형식 중 하나는 일시정지 광고입니다. 사용자가 검색 결과를 통해 발견한 비디오를 일시정지하면, 정지된 프레임 위에 맥락적 광고 오버레이가 나타납니다. 이 형식은 사용자가 콘텐츠에 적극적으로 집중하고 있는——높은 참여의 순간에 주목을 끌며, 비침입적인 방식으로 브랜드 메시지를 전달합니다.

TMG의 초기 테스트 데이터에 따르면, 일시정지 광고는 7.3퍼센트의 클릭률을 달성하고 표준 검색 결과 광고에 비해 2.1배 높은 브랜드 리콜을 생성합니다. 이 형식은 콘텐츠가 풍부한 검색 쿼리를 타겟하는 브랜드 인지도 캠페인에 특히 효과적입니다.

인피드 검색 광고 (2026년 신규)

인피드 검색 광고는 사용자가 검색 결과를 클릭한 후 나타나는 콘텐츠 피드에 원활하게 녹아듭니다. 별도의 검색 배치로 나타나는 대신, 이 광고들은 Bilibili가 선택한 비디오와 함께 표시하는 관련 콘텐츠 추천에 짜여집니다. 이 네이티브 통합은 인피드 검색 광고를 플랫폼에서 가장 방해가 적고 참여도가 높은 광고 형식 중 하나로 만듭니다.

Bilibili의 광고 팀에 따르면, 인피드 검색 광고는 평균 참여율 6.8퍼센트를 달성하며, 이는 플랫폼의 다른 모든 인피드 광고 형식보다 높습니다.

검색 제안 광고

검색 제안 광고는 사용자가 쿼리 입력을 시작할 때 Bilibili의 자동 완성 드롭다운에 나타납니다. 이 광고들은 사용자의 검색 여정의 가장 초기 단계에서 인터셉트하여, 브랜드에게 사용자가 쿼리를 완료하기 전에 검색 행동을 리다이렉트할 기회를 제공합니다. 이 형식은 경쟁사 트래픽을 포착하고 신제품 론칭에 대한 인지도를 높이는 데 특히 효과적입니다.

주요 사양:
- 자동 완성 제안에 표시
- 브랜드명과 짧은 슬로건 지원
- 평균 클릭률: 9.1퍼센트
- 일별 예산 한도가 있는 클릭당 비용 모델

맥락적 콘텐츠 광고

맥락적 콘텐츠 광고는 특정 콘텐츠 테마나 카테고리와 일치하는 검색 결과 옆에 표시됩니다. 개별 키워드를 타겟팅하는 대신, 이 광고들은 검색 쿼리의 맥락적 의미를 타겟팅하여 브랜드 메시지가 관련 콘텐츠 환경에 나타나도록 보장합니다. 이 형식은 Bilibili의 고급 자연어 처리 기능을 활용하여 사용자 쿼리의 의미적 의도와 광고를 매칭합니다.

Bilibili 검색 광고 타겟팅 옵션

Bilibili의 광고 플랫폼은 광고주가 검색을 통해 적절한 오디언스에 도달할 수 있도록 포괄적인 타겟팅 옵션 제품군을 제공합니다. 다음은 각 타겟팅 기능에 대한 자세한 설명입니다.

관심사 기반 타겟팅

Bilibili는 콘텐츠 소비 패턴, 참여 행동, 표현된 선호도에 기반하여 사용자를 200개 이상의 관심사 세그먼트로 분류합니다. 광고주들은 이러한 관심사 세그먼트와 검색 광고를 연결하여, 광고된 제품이나 서비스와 일치하는 입증된 관심사를 가진 사용자에게 메시지가 도달하도록 할 수 있습니다.

TMG는 관심사 기반 타겟팅을 검색 광고 전달을 정제하기 위한 넓은 필터로 사용할 것을 권장하며, 특히 관련 없는 트래픽을 유인할 위험이 있는 넓은 키워드 포트폴리오를 가진 캠페인에 적합합니다.

행동 타겟팅

Bilibili에서의 행동 타겟팅은 단순한 관심사 카테고리를 넘어 사용자의 플랫폼에서의 최근 행동을 고려합니다. 여기에는 다음이 포함됩니다:

  • 검색 기록: 최근 관련 용어를 검색한 사용자를 타겟팅합니다.
  • 시청 패턴: 비디오 소비가 구매 고려를 나타내는 사용자에게 도달합니다.
  • 참여 시그널: 관련 콘텐츠에 좋아요, 공유, 코인 투척, 북마크한 사용자를 타겟팅합니다.
  • 구매 행동: 이커머스 광고주를 위해, 같은 카테고리의 제품을 이전에 구매한 사용자를 타겟팅합니다.

인구통계 타겟팅

Bilibili는 광고주가 다음과 같은 인구통계 매개변수로 검색 광고 전달을 필터링할 수 있게 합니다:

  • 연령: 특정 연령 범위를 타겟팅하며, 특히 18~25세와 25~35세 코호트에 대해 세분화된 옵션이 있습니다.
  • 성별: 남성, 여성, 또는 모든 성별로 필터링합니다.
  • 지역: 도, 시, 또는 도시 등급(1등급, 2등급, 3등급+)으로 타겟팅합니다.
  • 기기: 기기 유형(모바일, 태블릿, 데스크톱)과 특정 기기 브랜드로 필터링합니다.

유사 오디언스 확장

기존 고객 데이터나 Bilibili 팔로워 기반을 보유한 광고주에게, 유사 오디언스 타겟팅은 알려진 오디언스를 넘어선 확장을 가능하게 합니다. Bilibili의 알고리즘은 시드 오디언스의 프로필 및 행동과 유사한 사용자를 식별하여, 브랜드가 기존 최고의 고객과 특성을 공유하는 새로운 잠재 고객에게 도달할 수 있게 합니다.

TMG의 테스트에 따르면, 검색 캠페인의 유사 오디언스는 유사한 전환 품질을 유지하면서 넓은 관심사 타겟팅에 비해 35퍼센트 낮은 획득 단가를 달성합니다.

맥락적 콘텐츠 타겟팅

Bilibili의 고유한 타겟팅 기능은 맥락적 콘텐츠 타겟팅으로, 광고주가 특정 콘텐츠 유형의 맥락에서 검색 광고를 배치할 수 있게 합니다. 예를 들어, 스킨케어 브랜드는 사용자가 뷰티 및 스킨케어 콘텐츠 카테고리 내에서 검색할 때만 검색 광고가 나타나도록 타겟팅할 수 있어, 광고 메시지와 사용자의 콘텐츠 소비 맥락 사이의 최대 관련성을 보장합니다.

Bilibili 검색 광고 최적화 전략

검색 광고를 시작하는 것은 시작에 불과합니다. 강력한 성과를 달성하려면 여러 차원에 걸친 지속적인 최적화가 필요합니다.

키워드 전략

잘 구조화된 키워드 전략은 Bilibili에서 성공적인 검색 광고의 기반입니다. TMG는 다음을 권장합니다:

  • 넓게 시작한 후 좁히기: 넓은 범위의 관련 키워드로 시작하고, 검색 쿼리 보고서를 사용하여 최고의 성과를 내는 용어를 식별합니다.
  • 매치 유형 레이어링: 정확한 매치, 구문 매치, 넓은 매치의 조합을 사용하여 도달과 정밀도의 균형을 맞춥니다.
  • 부정 키워드 모니터링: 검색 쿼리 보고서를 정기적으로 검토하여 관련 없는 쿼리를 식별하고 부정 키워드로 추가하여 낭비를 방지합니다.
  • 키워드 트렌드 추적: Bilibili의 검색 환경은 콘텐츠 트렌드와 함께 빠르게 진화합니다. 매주 새로운 키워드를 모니터링하고 입찰을相应 조정합니다.

크리에이티브 최적화

Bilibili에서의 검색 광고 크리에이티브는 빠르게 주목을 끌고 가치를 명확하게 전달해야 합니다. TMG의 크리에이티브 최적화 가이드라인은 다음을 포함합니다:

  • 가장 강력한 비주얼로 시작: Bilibili는 비디오 퍼스트 플랫폼입니다. 매력적인 비디오 썸네일을 특징으로 하는 검색 광고는 정적 이미지 광고에 비해 평균 43퍼센트 더 나은 성과를 냅니다.

Bilibili 검색 광고 성과 측정

Bilibili는 검색 광고주를 위한 강력한 측정 도구 세트를 제공합니다. TMG는 캠페인 효과를 평가하기 위해 다음과 같은 지표를 추적할 것을 권장합니다:

  • 노출수: 검색 광고가 표시된 총 횟수.
  • 클릭률 (CTR): 클릭으로 이어진 노출의 비율.
  • 클릭당 비용 (CPC): 각 클릭에 대해 지불된 평균 비용.
  • 전환율: 원하는 행동(구매, 등록, 다운로드 등)으로 이어진 클릭의 비율.
  • 획득당 비용 (CPA): 총 비용을 전환 수로 나눈 것.
  • 광고 지출 대비 수익 (ROAS): 검색 광고에 지출된 1위안당 생성된 매출.
  • 검색 인상 점유율: 광고가 포착한 총 사용 가능한 인상의 비율.
  • 품질 점수: 광고 관련성, 예상 클릭률, 랜딩 페이지 경험을 평가하는 Bilibili의 내부 지표.
