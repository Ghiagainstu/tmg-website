더우인에서 SEM vs 검색 피드: 어느 쪽이 더 높은 ROI를 제공하는가?

더우인의 브랜드 광고주들은 근본적인 전략적 질문에 직면합니다. 전통적인 키워드 타겟팅 SEM에 투자해야 할까요, 아니면 더우인의 알고리즘 기반 검색 피드 광고에 집중해야 할까요? 답은 둘 중 하나가 아닙니다. 각 모델이 뛰어난 영역을 이해하고 최대 수익을 위해 조합하는 것이 핵심입니다. 본 기사에서 TMG는两种 광고 유형의 메커니즘, 성과 데이터, 최적화 전략을 분석하여 검색 예산을 정밀하게 배분할 수 있도록 지원합니다.

SEM: 키워드 타겟팅 CPC 입찰 검색 광고

더우인의 SEM 모델은 전통적인 검색 엔진과 유사하게 작동합니다. 광고주가 특정 키워드를 선택하고, 클릭당 비용(CPC) 입찰을 설정하면, 사용자가 해당 키워드 또는 관련 용어를 검색할 때 광고가 표시됩니다. 광고주는 키워드 선택, 매칭 유형, 입찰가에 대한 직접적인 통제권을 유지합니다.

주요 특징:
- 타겟팅: 완전 일치, 구문 일치, 확장 일치 키워드.
- 입찰: 선택적 자동 입찰 조정이 포함된 수동 CPC.
- 배치: 검색 결과 페이지, 입찰 × 품질 × 관련성으로 배치 결정.
- 최적 용도: 광고주가 사용자가 원하는 것을 정확히 알고 있는 고의도, 명확한 검색 쿼리.

검색 피드: AI 매칭, oCPM 최적화 광고

더우인의 검색 피드 광고는 검색 광고에 대한 플랫폼의 차세대 접근법을 대표합니다. 명시적 키워드 타겟팅에 의존하는 대신, 알고리즘은 AI를 사용하여 사용자의 검색 행동, 콘텐츠 소비 패턴, 구매 이력을 기반으로 광고를 매칭합니다. 최적화는 원시 클릭이 아닌 다운스트림 전환 이벤트를 향해 입찰하는 oCPM(최적화 천회 노출 비용)을 중심으로 합니다.

주요 특징:
- 타겟팅: 행동 및 컨텍스트 시그널 기반 AI 주도 오디언스 매칭.
- 입찰: 특정 전환 목표(구매, 가입, 장바구니 담기)에 최적화된 oCPM.
- 배치: 유기적 콘텐츠와 함께 검색 결과 피드에 통합.
- 최적 용도: 광범위한 키워드 목록을 관리하지 않고 대규모로 수요를 포착하고자 하는 브랜드.

핵심 인사이트

SEM은 적극적인 검색자를 포착하고, 피드 광고는 수동적인 발견자에게 도달합니다. 최대 커버리지를 위해 둘 다 활용하세요.

🔍 성과 비교: SEM vs 검색 피드

TMG는 2026년 1분기 15개 상품 카테고리에 걸친 120개 이상의 캠페인에서 성과 데이터를 분석했습니다. 결과는 각 모델의 뚜렷한 강점을 드러냅니다.

클릭률(CTR)

  • SEM: 평균 CTR 3.2%. 명시적 키워드 매칭이 광고와 쿼리의 높은 관련성을 만들어 강력한 클릭 성과를 냅니다.
  • 검색 피드: 평균 CTR 2.1%. 더 넓은 AI 매칭이 때때로 부수적 쿼리에 광고를 표시하여 클릭 관련성을 약간 희석합니다.

전환율(CVR)

  • SEM: 평균 전환율 2.8%. 높은 의도가 견고한 전환으로 이어지지만, 좁은 키워드 초점이 총 볼륨을 제한합니다.
  • 검색 피드: 평균 전환율 4.1%. AI의 고확률 사용자 식별 능력 — 덜 명확한 쿼리에서도 — 이 우수한 전환 효율을 만들어냅니다.

고객 획득 비용(CPA)

  • SEM: 평균 CPA 85위안. 수동 CPC 입찰은 경쟁이 치열한 키워드에서 과다 입찰로 이어질 수 있습니다.
  • 검색 피드: 평균 CPA 62위안. oCPM 최적화가 가장 비용 효율적인 전환 경로를 향해 자동으로 입찰을 조정하여 낭비를 줄입니다.

광고 수익률(ROAS)

  • SEM: 평균 ROAS 3.8배. 신뢰할 수 있지만 키워드 볼륨과 수동 최적화 한계에 의해 제약됩니다.
  • 검색 피드: 평균 ROAS 5.2배. 알고리즘 접근법이 더 효율적으로 스케일링하며, 수동 키워드 선택이 놓칠 전환 기회를 식별합니다.

규모와 도달

  • SEM: 광고주 목록의 키워드 수와 해당 용어의 검색 볼륨에 의해 제한됩니다.
  • 검색 피드: 명시적 키워드 타겟팅 없이 AI가 관련 검색 쿼리에 광고를 매칭하므로 사실상 무제한 도달.

⚙️ oCPM 최적화의 역할

oCPM은 검색 피드 성과를 구동하는 엔진입니다. 클릭을 최적화하는 전통적인 CPC 입찰과 달리, oCPM은 광고주의 실제 비즈니스 목표(구매, 가입, 앱 설치)를 향해 최적화합니다.

더우인에서 oCPM 작동 방식

  1. 광고주가 전환 이벤트(예: "구매 완료")와 목표 CPA를 정의합니다.
  2. 더우인 알고리즘은 과거 전환 데이터와 실시간 사용자 시그널을 사용하여 각 노출이 원하는 전환으로 이어질 확률을 예측합니다.
  3. 시스템은 각 노출의 유효 입찰가를 자동으로 조정하여, 고확률 사용자에게는 높게, 저확률 사용자에게는 낮게 입찰합니다.
  4. 시간이 지남에 따라 알고리즘이 학습하고 예측을 정교화하여 효율성을 지속적으로 개선합니다.

TMG 데이터에 따르면 oCPM 캠페인은 학습 단계를 벗어나기 위해 주당 약 50건의 전환이 필요합니다. 최적화된 후 이 캠페인은 동일한 수동 CPC 캠페인 대비 일관되게 20~35% 낮은 CPA를 달성합니다.

oCPM vs 수동 CPC: 사용 시점

  • oCPM을 사용해야 할 때: 충분한 전환량(주 50건 이상), 명확한 전환 추적이 있고, 알고리즘이 가장 효율적인 경로를 찾기를 원할 때.
  • 수동 CPC를 사용해야 할 때: 검색 볼륨이 낮은 니치 카테고리에 있거나, 정밀한 키워드 수준 제어가 필요하거나, 완전 일치 정밀도가 중요한 브랜드 방어 캠페인을 실행할 때.

🔍 TMG의 통합 접근법: SEM과 검색 피드의 결합

더우인에서 가장 정교한 광고주들은 SEM과 검색 피드 사이에서 선택하지 않습니다. 각각의 강점을 활용하는 통합 전략으로两者를 결합합니다.

70/30 프레임워크

대부분의 상품 카테고리에 대한 TMG 권장 배분:

  • 70%를 검색 피드에: 규모, 효율, 알고리즘 최적화를 위해. 가장 낮은 CPA로 가장 넓은 수요를 포착합니다.
  • 30%를 SEM에: 정밀성, 브랜드 방어, 고의도 키워드 포착을 위해. 가장 중요한 브랜드 용어와 카테고리 용어에서의 가시성을 보장합니다.

이 비율은 카테고리 성숙도에 따라 조정됩니다:
- 신규 상품 카테고리: 발견 극대화를 위해 검색 피드 우선 80/20.
- 성숙한 경쟁 카테고리: 고의도 키워드 방어를 위해 SEM 우선 60/40.

키워드 시너지

TMG는 SEM 키워드 성과 데이터를 검색 피드 최적화에 활용합니다. 특정 키워드가 SEM에서 강력한 성과(높은 CTR, 높은 CVR)를 보이면, 해당 타겟팅 시그널을 검색 피드 캠페인으로 확장하여 어떤 유형의 쿼리가 가장 잘 전환되는지를 알고리즘에 효과적으로 가르칩니다. 반대로, 검색 피드 데이터가 고성과 오디언스 세그먼트를 드러내면, 대응하는 SEM 키워드 캠페인을 만들어 해당 수요를 정밀하게 포착합니다.

크리에이티브 얼라인먼트

两种 모델은 동일한 크리에이티브 풀을 공유하지만 최적화 우선순위가 다릅니다:
- SEM 크리에이티브는 키워드 관련성과 명확한 상품 차별화를 우선시합니다.
- 검색 피드 크리에이티브는 광범위한 어필과 알고리즘 친화성 — 다양한 오디언스 세그먼트에서 고성과를 내는 후크를 우선시합니다.

TMG는 일반적으로 SEM에는 3~4개의 크리에이티브 변형(키워드별), 검색 피드에는 5~7개의 변형(오디언스 다양)을 운영하며 2주 주기로 갱신합니다.

🏷️ TMG 사례 연구: 전자제품 브랜드 ROI 전환

한 소비자 전자제품 브랜드가 검색 예산의 100%를 수동 CPC SEM에 지출하며 2.4배의 ROAS를 달성하고 있었습니다. TMG는 60일에 걸쳐 배분을 검색 피드 65% / SEM 35%로 재구조화했습니다.

성과:
- ROAS가 2.4배에서 4.7배로 개선.
- CPA가 112위안에서 68위안으로 감소.
- 예산 증가 없이 총 전환이 89% 증가.
- 검색 피드 구성요소가 기존 SEM 키워드 목록에 없던 신규 오디언스 세그먼트(35~44세 테크 매니아)를 발견.

📄 핵심 요약

  • SEM은 더 높은 CTR과 정밀한 키워드 제어를 제공하지만 규모와 효율에 제한이 있습니다.
  • 검색 피드는 oCPM 최적화를 통해 더 높은 CVR, 낮은 CPA, 우수한 ROAS를 제공합니다.
  • 최적 전략은两者를 결합하는 것입니다: 규모를 위한 검색 피드 70%, 정밀성을 위한 SEM 30%.
  • oCPM 캠페인은 학습 단계를 벗어나 최고 성과를 내려면 주 50건 이상의 전환이 필요합니다.
    -两种 모델 간의 키워드 및 크리에이티브 시너지가 전체 결과를 증폭합니다.
