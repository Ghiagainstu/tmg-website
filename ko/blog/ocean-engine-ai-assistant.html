메이트 즈싱: 오션 엔진의 AI 광고 어시스턴트

Ocean Engine의 AI 광고 어시스턴트 Zhitu Xing(智图星)은 스마트 Q&A, 계정 분석, 데이터 쿼리, 진단 분석 기능을 플랫폼 내에서 제공합니다.

📄 Zhitu Xing이란?

Zhitu Xing은 Ocean Engine 앱(ByteDance의 Douyin 광고 캠페인 관리 모바일 플랫폼)에 네이티브로 내장된 AI 어시스턴트입니다. 주요 플랫폼 업데이트의 일환으로 출시되었으며, 광고 제품 전반에 생성형 AI를 통합하려는 ByteDance의 전략을 반영합니다.

어시스턴트는 홈 화면의 플로팅 버튼과 앱 전반의 컨텍스트 진입점을 통해 접근할 수 있습니다. 서드파티 분석 도구나 외부 챗봇과 달리, Zhitu Xing은 계정 데이터에 직접 접근하여 캠페인 성과를 읽고, 문제를 진단하며, 추천을 제공합니다. CSV 파일을 내보낼 필요가 없습니다.

플랫폼 참고

Zhitu Xing은 Ocean Engine 웹 대시보드가 아닌 Ocean Engine 앱(巨量引擎APP)에서만 사용할 수 있습니다. 앱은 iOS와 중국 앱 스토어에서 다운로드 가능하며, 인증된 에이전시의 QR 코드 스캔을 통해서도 접근할 수 있습니다.

⚙️ 4가지 핵심 기능

ByteDance는 공식 제품 매뉴얼에서 Zhitu Xing의 4가지 주요 사용 사례를 문서화했습니다. 각각 광고 워크플로우의 다른 레이어를 다룹니다.

1
스마트 Q&A (智能问答)

텍스트나 음성으로 광고 관련 질문을 하면 Zhitu Xing이 이해하기 쉬운 언어로 응답합니다. 도움말 문서를 검색할 필요 없이 질문 하나에 답변 하나.

2
계정 분석 (单/多账户解读)

하나 또는 여러 계정의 거시적 성과를 요약합니다. 이상치를 감지하고, 상위·하위 성과 계정을 식별하며, 업계 데이터와 벤치마킹합니다.

3
스마트 데이터 쿼리 (智能问数)

자연어로 캠페인 데이터를 쿼리합니다. 시간, 광고 단위, 크리에이티브 ID, 목표별 필터링과 지출, CTR, CVR, ROI 등의 지표를 지원합니다.

4
진단 분석 (诊断分析)

저성과 크리에이티브와 광고를 식별하고, 비효율적 지출의 원클릭 정리, 거부된 크리에이티브의 원클릭 수정 후 재게시를 지원합니다.

🤖 1. 스마트 Q&A — 인라인 지식 베이스

스마트 Q&A 기능은 질문을 입력하거나 말하면 구조화된 쉬운 언어의 응답을 제공합니다. Ocean Engine 도움말 센터를 검색하거나, 계정 관리자에게 묻거나, 지원 티켓 답변을 해석할 필요가 없습니다.

실무에서는 운영 질문에 가장 유용합니다: 광고가 심사 중인 이유, 이 지표의 의미, 리타겟팅 오디언스 설정 방법, oCPX와 CPC 입찰의 차이점 등. 어시스턴트는 맥락이 풍부한 질문을 이해하고 적절한 수준의 상세함으로 답변합니다.

감사 관련 이슈와 인간 검토 에스컬레이션의 경우, ByteDance는 별도 시스템인 Juliangwenwen(巨量问问)으로 라우팅합니다. Zhitu Xing은 일반 지식과 계정 질문을 처리하고, Juliangwenwen은 컴플라이언스와 심사 상태를 처리합니다.

📊 2. 계정 분석 — 모든 계정을 하나로 요약

여러 광고주 계정을 관리하는 경우(대부분의 에이전시와 대형 브랜드가 해당), 다중 계정 분석 기능이 특히 유용합니다. 포트폴리오 전반의 현황과 문제 집중 영역을 통합 뷰로 제공합니다.

이 기능은 성과 요약, 이상치 감지, 업계 벤치마크 비교를 제공합니다. 여러 브랜드나 지역을 관리하는 에이전시나 대형 광고주에게 특히 가치 있습니다.

📊 3. 스마트 데이터 쿼리 — 데이터와 대화하세요

자연어 쿼리를 데이터 요청으로 번역하여 기술적 SQL이나 플랫폼 필터를 몰라도 성과 데이터를 검색할 수 있게 합니다.

시간, 광고 단위, 크리에이티브 ID, 목표별 필터링과 지출, CTR, CVR, ROI 등의 지표를 지원합니다. 여러 광고 계정에 걸친 성과를 단일 대시보드 뷰로 집계합니다.

⚠️ 4. 진단 분석 — 문제 발견 및 수정

저성과 크리에이티브와 광고를 식별하고, 비효율적 지출의 원클릭 정리를 지원합니다. 거부된 크리에이티브의 원클릭 수정 후 재게시 기능도 제공합니다.

성과 추천은 AI 생성이며, 인간의 판단을 대체하지 않습니다. 중요한 예산 결정에는 항상 검증이 필요합니다.
