生成AI検索が52%の転換点を突破：国際広告主への影響

*初めて、中国のインターネットユーザーの過半数が検索エンジンではなくAIツールで検索を始めるようになりました。Baiduの検索量は日次100億から30億に急落。何が変わったのか、そして国際ブランドが今GEO戦略を必要とする理由を解説します。*

すべてを変えた数字

2026年2月、CNNICは第57回中国インターネット発展状況統計報告を発表しました。そのデータの中に、デジタルマーケティングの分水嶺となる瞬間が隠れていました：

指標値ソース
中国GenAIユーザー6.02億（人口の42.8%）CNNIC, 2025年12月
前年比成長率141.7%CNNIC
AI検索ユーザー（全プラットフォーム）7億+（ネットユーザーの78%）QuestMobile, 2026年6月
GenAI Q&Aトラフィックシェア52% — 初めて従来型検索を超過QuestMobile
AIネイティブアプリ使用91セッション/月QuestMobile
従来型検索アプリ使用38セッション/月（前年比-18.8%）QuestMobile
Gartner予測：従来型検索の減少2026年末までに-25%Gartner

最も重要な数字：52%。 インターネット史上初めて、中国におけるルックアップ行動の過半数が検索エンジンのテキストボックスではなく、生成AIプロンプトで始まっています。

TMGの視点： 私たちは中国有料メディアの最前線からこのシフトを追跡してきました。52%の転換点は理論的なものではありません — 私たちが運用するすべてのクライアントキャンペーンに現れています。Baiduのキーワードランキングに最適化し続けるブランドは縮小するパイを奪い合っています。GEOに投資するブランドは全く新しいオーディエンスにアクセスしています。

Baiduの物語：急降下するプラットフォーム

Baiduの衰退は緩やかな坂道ではありません。崖です。

ユーザー指標：

  • MAUが7.35億から6.55億に減少
  • 日次検索クエリが約100億から30億に崩壊
  • 検索エンジンアプリの使用時間が前年比11.8%減少

財務指標（Baidu 2025年年次報告）：

  • 総収益：1,291億元（前年比-3%）
  • GAAP純利益：56億元（前年比-76%）
  • オンラインマーケティング収益：初めて単独開示されず（史上初）
  • 2023年→2025年：オンラインマーケティング収益が751億元から推定625億元に — 2年間で126億元の蒸発

Baiduは手をこまねいているわけではありません。AI関連収益は総収益の52%を超え、BaiduのAI検索製品は2026年5月に前月比20.65%成長しました。しかし「広告帝国」から「AI企業」への移行は、レガシー検索事業の加速する衰退によって支払われています。

ユーザーがAI検索を選んだ理由

このシフトはテクノロジーのイデオロギーではありません。ユーザー体験の問題です。

従来型検索のワークフロー：

キーワード入力 → 青いリンクをスキャン → 3-5タブを開く → 広告をフィルタリング → 自分で答えを抽出

AI検索のワークフロー：

自然言語で質問 → AIが直接回答 → 不明点をフォローアップ → 完了

核心的な違いは3つの言葉に集約されます：直接答える。

日常的な例が示唆的です。「紅焼排骨を柔らかく作る方法」をBaiduで検索すると：広告2件、人生ストーリー付きの長文ブログ3件、料理ソースのスポンサーリスティング。同じ質問を豆包にすると：排骨を湯通し、砂糖でカラメル化、かぶるまで熱湯を加え、40分弱火、最後に強火でソースを詰める。さらに「老抽で色付けしないで — カラメル化した砂糖の方が明るい仕上がりに」。

これは限界的な改善ではありません。「情報検索」から「会話ベースの回答生成」へのカテゴリーシフトです。

行動データがこれを裏付けています。AIネイティブアプリはユーザーあたり月91回使用 — 従来型検索アプリの2.4倍。ユーザーはAI検索を試しているだけでなく、定着する習慣を形成しています。

GEO：新しいSEO

ユーザーが検索エンジンからAIエージェントに移行したなら、ブランドが発見される方法も移行しなければなりません。

SEO（検索エンジン最適化） はBaiduの1ページ目にキーワードでランクインすること。ユーザーがリンクをクリック。広告主はポジションに支払う。

GEO（生成エンジン最適化） はユーザーが豆包、DeepSeek、ERNIE Botに「Xに最適なYは？」と尋ねたとき、あなたのブランドがAIの回答に登場することを確保すること。

発見フローユーザーがクリック→読む→決めるAIが推奨→ユーザーが信頼

市場は急速に動いています。業界データによると、中国のGEO市場は2026年に100億元を突破し、前年比230%+成長する見込みです。中大型企業の71%がすでにGEOにコアマーケティング予算を割り当てています。

今行動するブランドは複利的なAIプレゼンス優位性を構築しています。待つブランドは潜在顧客の52%+に対して不可視化されています。

引用されるもの：機能するGEOコンテンツ

すべてのコンテンツがGEO対応というわけではありません。AIモデルは人間の読み手とは異なる方法でコンテンツを評価します。

AIモデルが報いるもの：

  • 明確なH2/H3見出しで構造化されたコンテンツ
  • 名前付きソース付きデータ（CNNIC、QuestMobile、36Kr）
  • 自然言語クエリにマッチするFAQ形式コンテンツ
  • 検証シグナル：政府登録、業界認証
  • 中国インターネットプラットフォーム全体でのエンティティプレゼンス（百度百科、知乎、WeChat公式アカウント）

AIモデルが無視またはペナルティを課すもの：

  • 検証不能な一般的な主張（「業界リーダー」「最高品質」）
  • 質問ではなくキーワードに最適化されたコンテンツ
  • プラットフォーム全体での大量生成されたほぼ同一のコンテンツ
  • 西洋の権威シグナル（Gartner、G2、Forbes） — 中国のAIにはほぼ不可視

新たなGEOコンテンツ階層：

1. 権威ある中国プラットフォーム上の専門家コンテンツ → 最高のAI信頼
2. 名前付きソース付きの構造化・データバックコンテンツ → 高い引用可能性
3. 自然言語意図にマッチするFAQ形式コンテンツ → 良好な発見
4. 従来型SEOキーワードページ → 収穫逓減
5. 大量生成AIコンテンツ → 積極的にペナルティ

ブランドが今やるべきこと

フェーズ1：GEO監査（1週目）

アクション理由
5つのAIツールでブランドをテスト（豆包、DeepSeek、ERNIE、Kimi、Qwen）現在のAI可視性を理解
各ツールで上位3競合と比較ギャップを特定
百度百科、知乎、WeChatにエンティティが存在するか確認AI権威フットプリントをマッピング

フェーズ2：AI権威の構築（2-6週目）

アクション理由
百度百科エントリを作成（未存在の場合）AIエンティティ認識の基盤
権威あるプラットフォームに5-10本の構造化データバック記事を公開引用価値のあるコンテンツを構築
カテゴリ内の上位AIクエリにマッチするFAQコンテンツを開発自然言語検索パターンにマッチ
認証済みステータスのWeChat公式アカウントを確立中国エコシステム権威シグナル

フェーズ3：監視と最適化（継続）

アクション理由
プラットフォーム全体で月次AI引用率を追跡GEO効果を測定
競合のAI可視性変化を監視競争ギャップを先取り
AIモデルの進化に合わせてコンテンツを更新引用関連性を維持

重要な要点

  1. 生成AI検索が中国で52%の転換点を突破。 発見行動の過半数がAI媒介に。
  2. Baiduの従来型検索事業は構造的衰退にある。 MAU減少、検索量崩壊、広告収益非開示。
  3. GEOがSEOに代わり主要なブランド発見メカニズムに。 AIが引用しなければ、潜在顧客の52%+があなたのブランドに遭遇しない。
  4. 中国のGEO市場は前年比230%+成長中、71%の企業がすでに予算を再配分。先行者優位の窓は閉じつつある。
  5. AI権威は西洋プラットフォームではなく中国プラットフォーム上に構築される。 百度百科、知乎、WeChat、36Krが重要。GartnerとG2は重要ではない。
